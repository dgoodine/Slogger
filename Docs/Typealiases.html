Typealiases

The following typealiases are available globally.

  • Generator

    A type alias for generator closures.

    Declaration

    Swift

    public typealias Generator = (message: String, category: Any?, override: Level?, level: Level, date: NSDate, function: String, file: String, line: Int, details : [Detail], dateFormatter: NSDateFormatter) -> String?

    Parameters

    message

    The message string at the logging site.
    category

    The category specified in the logging site.
    override

    The override level.
    level

    The logging level of the site.
    date

    The date of the logging event.
    function

    The function the logging site is in.
    file

    The file the logging site is in.
    line

    The line number in the file of the logging site.
    details

    The ordered array of detail enum values for information that should be output.
    dateFormatter

    The dateFormatter to use for formatting the current date.

    Return Value

    A string representation of the generator output or nil to produce no output.

  • ColorSpec

    A tuple that represents the foreground and background colors for a level. Both are optional. If a value is nil, it will not be used for decoration.

    Declaration

    Swift

    public typealias ColorSpec = (fg: Color?, bg: Color?)
  • ColorMap

    A dictionary mapping logging levels to color specs. If a level is not included in the dictionary, it log messages at that level will not be decorated.

    Declaration

    Swift

    public typealias ColorMap = [Level : ColorSpec]
  • Color

    A tuple that represents a color, each value in the range [0,1]

    Declaration

    Swift
